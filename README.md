# Catálogo de Ropa Minimalista

Una página web minimalista para mostrar un catálogo de ropa con paleta de colores en blanco, negro y grises.

## Características

- Diseño minimalista y elegante
- Paleta de colores monocromática (blanco, negro, grises)
- Sistema de filtros por categoría
- Diseño responsive (adaptable a móviles y tablets)
- Interfaz limpia y moderna

## Estructura de Archivos

- `index.html` - Estructura principal de la página
- `styles.css` - Estilos minimalistas
- `script.js` - Funcionalidad de filtros y renderizado de productos

## Cómo Usar

1. Abre el archivo `index.html` en tu navegador web
2. Navega por las diferentes categorías usando los botones de filtro
3. Explora los productos del catálogo

## Personalización

Para agregar más productos, edita el array `productos` en el archivo `script.js`:

```javascript
{
    id: 13,
    nombre: "Nombre del Producto",
    categoria: "camisetas", // camisetas, pantalones, chaquetas, accesorios
    precio: "$XX.99",
    imagen: "nombre-imagen"
}
```

## Categorías Disponibles

- Camisetas
- Pantalones
- Chaquetas
- Accesorios
